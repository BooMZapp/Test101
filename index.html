เกี่ยวกับฉัน (About Me)

ผมเป็นนักพัฒนาเว็บไซต์ที่มีความเชี่ยวชาญด้าน Front-end และมีความเข้าใจในหลักการออกแบบ UI/UX เป็นอย่างดี ผมเชื่อว่าเว็บไซต์ที่ดีไม่ได้มีแค่โค้ดที่มีประสิทธิภาพ แต่ต้องมอบประสบการณ์ที่ดีที่สุดให้กับผู้ใช้งานด้วย

ทักษะของผมประกอบด้วย: HTML, CSS, JavaScript, React และการออกแบบด้วย Figma. ผมมีความสุขที่ได้เรียนรู้เทคโนโลยีใหม่ๆ และนำมาปรับใช้เพื่อสร้างผลงานที่ดียิ่งขึ้นเสมอ และพร้อมที่จะเรียนรู้สิ่งใหม่ๆ เพื่อพัฒนาตัวเองอยู่เสมอ

หากคุณกำลังมองหานักพัฒนาเว็บไซต์ที่ใส่ใจในรายละเอียด และพร้อมสร้างสรรค์ผลงานคุณภาพ ผมยินดีที่จะเป็นส่วนหนึ่งในโปรเจกต์ของคุณครับ!

ผลงานของฉัน (Portfolio)

ผลงานที่ 1: ได้รับรางวัล“คุรุชน คนคุณธรรม”

โปรเจกต์นี้เป็นการออกแบบและพัฒนาเว็บไซต์ E-commerce ที่เน้นประสบการณ์ผู้ใช้ที่ดี (UX) และการแสดงผลที่สวยงาม (UI) ด้วย React และ styled-components

ผลงานที่ 2: ระบบจัดการเนื้อหา (CMS)

พัฒนาระบบหลังบ้านสำหรับจัดการเนื้อหาด้วย Node.js และ Express.js พร้อมฐานข้อมูล MongoDB เพื่อความยืดหยุ่นในการจัดการข้อมูล

ผลงานที่ 3: แอปพลิเคชัน Progressive Web App (PWA)

สร้าง PWA สำหรับการจองบริการออนไลน์ ให้ผู้ใช้สามารถเข้าถึงแอปได้แม้ไม่มีอินเทอร์เน็ต ด้วย Vue.js และ Service Workers

ผลงานที่ 4: Landing Page สำหรับ Start-up

ออกแบบ Landing Page ที่น่าดึงดูดและปรับแต่งได้ง่าย สำหรับโปรโมทผลิตภัณฑ์ใหม่ของ Start-up เน้นการแปลงผู้เข้าชมให้เป็นลูกค้า
